# Screenshots das Calculadoras

Este diretório contém as imagens usadas no hero carousel da landing page.

## Imagens Necessárias

Atualmente usando placeholders. Para melhores resultados, substitua por screenshots reais:

1. **margem-lucro.png** (800x600px)
   - Screenshot da calculadora de Margem de Lucro
   - Deve mostrar a interface preenchida com dados de exemplo

2. **preco-hora.png** (800x600px)
   - Screenshot da calculadora de Preço por Hora
   - Deve mostrar a interface com um cálculo completo

3. **precificacao.png** (800x600px)
   - Screenshot da calculadora de Precificação
   - Deve mostrar produtos/serviços precificados

4. **das.png** (800x600px)
   - Screenshot do Calendário DAS
   - Deve mostrar o calendário do mês atual

## Como Gerar

1. Acesse cada calculadora no navegador
2. Preencha com dados de exemplo realistas
3. Capture screenshot em resolução 800x600
4. Otimize com TinyPNG ou ImageOptim
5. Salve neste diretório

## Observações

- Use dados fictícios mas realistas
- Evite informações pessoais ou reais
- Mantenha interface limpa e organizada
- Screenshots devem funcionar em light e dark mode
